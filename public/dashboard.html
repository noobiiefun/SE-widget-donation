Saweria
Webhook + Stream Key verification
Stream Key (untuk verifikasi signature)

1. Buka saweria.co/dashboard → Integrasi → Webhook
2. Masukkan URL webhook kamu di kolom Webhook URL (butuh ngrok — lihat panduan bawah)
3. Contoh: https://abc123.ngrok.io/webhook/saweria
4. Copy Stream Key dari halaman yang sama → paste di atas

🩷
Trakteer
Webhook (butuh URL publik)

1. Buka trakteer.id/manage → Integrasi → Webhook
2. Masukkan URL: https://abc123.ngrok.io/webhook/trakteer
3. Aktifkan webhook → klik Send Webhook Test untuk verifikasi
4. Tidak perlu token khusus — Trakteer tidak pakai signature

Streamlabs
Socket API — tidak butuh ngrok!
Socket API Token

1. Buka streamlabs.com/dashboard → API Settings
2. Tab API Tokens → Copy Socket API Token
3. Paste di atas → Simpan → Tidak perlu ngrok, server ini konek langsung ke Streamlabs! ✅

Setup Ngrok (Untuk Saweria, Trakteer, SociaBuzz)
Gratis — expose localhost ke internet agar webhook bisa masuk
1
Download ngrok gratis di ngrok.com/download → daftar akun gratis → install
2
Authenticate: ngrok config add-authtoken YOUR_TOKEN (token ada di dashboard ngrok setelah daftar)
3
Jalankan server ini dulu (npm start), lalu buka terminal baru dan ketik: ngrok http 3000
4
Ngrok akan menampilkan URL seperti https://abc123.ngrok.io — ini adalah alamat publik server kamu
5
Paste URL ngrok kamu di sini, lalu klik Generate Webhook URLs — akan muncul URL siap copy untuk tiap provider
