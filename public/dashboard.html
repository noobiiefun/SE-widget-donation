Cara Kerja
Notifikasi Saweria/Trakteer/dll → masuk ke overlay SE kamu
🔄
Alur: Provider donasi → kirim webhook ke server ini → server push ke SE Custom Widget via SSE → notifikasi muncul di overlay SE kamu ✅
SE overlay kamu tetap utuh, widget ini cuma tambahan di atas overlay yang sudah ada.
1
Jalankan server ini — npm start atau double-click start.bat
2
Klik Copy Widget Code → SE di pojok kanan atas — ini kode yang di-paste ke SE
3
Di SE Overlay Editor → Add Widget → Custom Widget → Open Editor → tab HTML → hapus semua → paste kode
4
Tab CSS → kosongkan. Tab JS → kosongkan. Tab Fields → kosongkan. Klik Done → Save
5
Atur ukuran widget di canvas SE: Width 1920 × Height 1080, posisi X:0 Y:0 — full screen
6
Setup provider di bawah → klik Test Alert → notifikasi muncul di overlay SE kamu ✅

Setup Ngrok (Saweria, Trakteer, SociaBuzz)
Gratis — expose localhost ke internet agar webhook bisa masuk
1
Download ngrok.com/download → daftar akun gratis → install
2
Di terminal: ngrok config add-authtoken TOKEN_KAMU
3
Jalankan server ini dulu, lalu terminal baru: ngrok http 3000
4
Copy URL ngrok (contoh: https://abc123.ngrok.io) → paste di bawah
